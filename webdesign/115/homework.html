Web Designer

What is a Web Designer

A person who "writes HTML and CSS code and may fulfill some graphic designer job duties, such as determining the appropriate use of color, designing wireframes and page layouts, creating logos and graphics, and optimizing images for display on the Web (Feke-Morris)."
